# numguess

사용자로부터 입력 받은 유추값을 정답과 비교하는 프로그램

# Installation
``` shell
$ pip install
```

# How to Start
```shell
$ python numguess.py
```

# Features
- 사용자 이름 입력받기
- 사용자 유추값 입력받기
- 정답값 랜덤으로 생성하기
- 사용자의 유추값과 정답값 비교하기
